# Data Wrangling and Visualization

In this subsection, we import the data from NOAA, clean the data, and perform some preliminary exploration. `Data-Wrangling.ipynb` contains the import and cleaning, while `Data-Visualization.ipynb` contains exploratory graphs and summaries of the data in 2D and 3D (using PCA).
